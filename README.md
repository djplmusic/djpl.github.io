# 🎧 Site DJ - Réservation & Boutique

Bienvenue sur mon site officiel de DJ ! 🔥  
Ce site permet de découvrir mon univers, voir mes prestations et me contacter facilement pour des événements.

## 🚀 Fonctionnalités

- 📸 **Galerie** : Photos et présentations de mes événements
- 📩 **Contact** : Formulaire pour demander un devis ou me contacter
- 🎶 **Présentation** : Découverte de mon style et de mes prestations
- 🛒 **Boutique** : Produits disponibles à l'achat (merch, services, etc.)

## 🎯 Objectif

Offrir une plateforme simple et efficace pour :
- Présenter mon travail
- Attirer de nouveaux clients
- Faciliter les demandes de réservation

## 🛠️ Technologies utilisées

- CSS
- JavaScript
